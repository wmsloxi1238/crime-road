## 다운로드 안내

- data.go.kr / data.seoul.go.kr 파일은 **로그인 없이 "다운로드" 버튼**으로 받음 (브라우저 필요, JS 동작).
- API형(WMS/REST)은 **무료 활용신청 → 인증키** 발급 후 사용.
- 받은 CSV를 `~/Documents/cw-p` 에 넣으면 강남 행 추출·병합·지도화 진행.

## 분석 설계 (권장 결합)

1. **위험 레이어**: 범죄주의구간(15101797, 도로등급) + 교통사고 다발지(15029185)
2. **환경 레이어**: 가로등(15107934) 밀도 + CCTV(15116151) 위치 → "어둡고 CCTV 없는 골목"
3. **맥락 레이어**: 길단위 유동인구(OA-15568) + 보행 네트워크(OA-21208)
4. **결과**: 강남 골목별 위험점수 = f(범죄밀도, 사고, 조명부족, CCTV공백, 유동인구)
